# 💰 Finance Dashboard UI

A modern, interactive, and fully responsive finance dashboard built using React, Vite, and Tailwind CSS.
This project simulates a real-world financial tracking application with powerful analytics, role-based access, and a premium user interface.

---

## 🚀 Live Demo

[(Live Demo)](https://finance-dashboard-ten-flax.vercel.app/)

---

## 📌 Overview

The Finance Dashboard allows users to:

* Track financial transactions
* Visualize income and expenses
* Analyze spending patterns
* Manage data with role-based controls

This project focuses on **clean architecture, user experience, and frontend engineering best practices**.

---

## 🛠️ Tech Stack

* **Frontend:** React (Vite)
* **Styling:** Tailwind CSS
* **State Management:** Context API
* **Charts:** Recharts
* **Animations:** Framer Motion, GSAP
* **Data Persistence:** LocalStorage

---

## ✨ Features

### 📊 Dashboard Overview

* Summary cards (Total Balance, Income, Expenses)
* Clean and responsive layout

### 📈 Data Visualization

* Line chart for balance trend over time
* Pie chart for category-wise spending

### 📋 Transactions Management

* View all transactions in a structured table
* Add, edit, and delete transactions (Admin only)
* Search, filter, and sort functionality
* Date range filtering

### 🔐 Role-Based Access

* **Viewer:** Read-only access
* **Admin:** Full control (Add, Edit, Delete)

### 💡 Insights & Analytics

* Highest spending category
* Monthly expense comparison
* Smart insights based on user data

### 💾 Data Persistence

* Transactions saved using LocalStorage
* Data remains after page refresh

### 📤 Export Feature

* Export transactions as CSV

### 🎨 UI/UX Enhancements

* Dark/Light theme toggle
* Premium Apple-inspired UI design
* Responsive across all devices
* Smooth animations using Framer Motion & GSAP

### 🌐 Landing Page

* Modern landing page explaining features
* Smooth scroll animations
* Clean SaaS-style layout

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/sagarjain03/finance_dashboard
cd finance-dashboard
```

---


### 2. Install dependencies

```bash
npm install
```

---

### 3. Run the development server

```bash
npm run dev
```

---

### 4. Open in browser

```
http://localhost:5173
```
